Featured Project · CARS24

AI Voice Calling Platform

A scalable AI-assisted outbound calling workflow that scales lead engagement and appointment booking in the sell-car journey — reducing calling cost by 70%.

The Problem
Users showed clear intent on CARS24 — they checked their car's price and viewed a quote — but dropped off before booking a free inspection. Manual outbound calling was expensive, inconsistent, and hard to scale. Most calls followed the same script: confirm interest, ask for a preferred slot, collect the address, and book the appointment.

AI Vehicle Variant Matching

An OpenAI-powered fallback that resolves one internal CARS24 vehicle variant from ambiguous RC/VAHAN data when the data-science matching API returns multiple or no candidates.

"The model was constrained to choose only from a closed list of valid CARS24 variant IDs or return null, preventing hallucinated variant outputs."

Constrained disambiguation — cannot invent variant IDs
○ Year range valid (reg 2016 → [2014–2017])

○ Transmission type (AMT → AMT only)
○ Fuel type match (Petrol → Petrol)
